# 📺 Netflix Data Visualization Project

This project is a beginner-friendly data visualization exercise using a sample Netflix dataset. It showcases how to explore and present data through various chart types using *Matplotlib* and *Pandas* in Python.

---

## 📊 What This Project Includes

### ✅ Visualizations:
1. *Bar Chart* – Title vs Watch Hours  
2. *Pie Chart* – Distribution of Shows by Genre  
3. *Line Plot* – Release Year vs Rating  
4. *Scatter Plot* – Ratings vs Watch Hours

All four charts are arranged neatly using subplots() in a single figure for dashboard-style viewing.
